int EditarExpr ( char *  nomearq)

Edita a expressão que está dentro de um arquivo
Função que edita uma expressão matemática que está dentro um arquivo, recebe como parâmetro uma string com o nome desse arquivo e retorna um inteiro com o valor dessa expressão calculado pela função "leitura".

Parâmetros
nomearq- string que constitui o nome do arquivo a ser editado
Retorna
inteiro com o valor da expressão
Observações
Assertiva de entrada:
  • existir o arquivo passado como parâmetro
Assertiva de saída:
  • valor retornado é um inteiro EXIT_SUCCESS ou EXIT_FAILURE
Estrutura implícita:
  • Não são usadas variáveis globais, arquivos externos ou condições de retorno
Estrutura Explícita:
  • Recebe uma string e retorna um inteiro
Requesitos:
  • Deve ser capaz de imprimir a expressão original na tela
  • Deve capturar a nova expressão pelo usuário e posteriormente sobrescrever o arquivo com a nova expressão
Hipótese:
  • A expressão fornecida estará correta em quesitos não relativos à parentização ou lógica matemática
  • Será fornecida apenas uma expressão matemática e esta será não nula
