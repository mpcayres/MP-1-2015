int leitura ( char *  nomearq)

Testa a parentização dos elementos de uma pilha
Retorna a determinação de se uma expressão encontrada em um arquivo é válida (EXIT_SUCCESS, em caso positivo, e EXIT_FAILURE, em caso negativo) e exibe essa informação na tela, juntamente ao seu resultado.

Parâmetros
nomearq- ponteiro de char indicando a string correspondente ao nome do arquivo/código
Retorna
Um valor inteiro que serve como booleano
Observações
Assertiva de entrada:
  • existir o arquivo passado como parâmetro
Assertiva de saída:
  • valor retornado é um inteiro EXIT_SUCCESS ou EXIT_FAILURE
  • arquivo não sofrerá modificações
Estrutura implícita:
  • Não são usadas variáveis globais, arquivos externos ou condições de retorno
Estrutura Explícita:
  • Recebe como parâmetro uma string e retorna um inteiro
Requesitos:
  • Deve ser capaz de verificar a validade de uma expressão
  • Caso não haja encontrado o arquivo, cessar o procedimento
  • Liberar qualquer estrutura utilizada para auxiliar no procedimento
Hipótese:
  • A expressão fornecida estará correta em quesitos não relativos à parentização ou lógica matemática
  • O arquivo aberto pela string passada como parâmetro possui apenas uma expressão matemática
