Pilha* cria_pilha ( )

Criar uma pilha;
Aloca um espaço para pilha e faz o elemento 'topo' apontar para NULL, este que será o início da lista a ser utilizada.

Parâmetros
Nenhum
Retorna
A pilha alocada
Observações
Estrutura implícita:
  • Não é usada variáveis globais, arquivos externos ou condições de retorno
Estrutura Explícita:
  • Não recebe parâmetro, mas retorna uma pilha
Requesitos:
  • Deve ser capaz de alocar espaço para uma pilha
  • Caso não haja espaço na memória, cessar o procedimento
Hipótese:
  • O usuário irá usar uma vez a função antes de qualquer operação com a pilha
